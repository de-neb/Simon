SIMON

How to play?
  1. Press any button. Simon will give the first signal. Repeat the signal by clicking the same color.
  2. Simon will duplicate the first signal and add one. Repeat these two signals by clicking the same colors in order.
  3. Simon will duplicate these first two signals and add one.
  4. Continue playing as long as you can repeat each sequence of signals correctly.
  5. The game is over if you fail to repeat a sequence exactly.
